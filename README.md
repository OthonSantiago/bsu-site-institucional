# BSU Site Institucional

Site institucional da Banda Sinfônica da Unicamp, com foco em identidade cultural, agenda de concertos, corpo artístico, imprensa, contato, booking e apoio cultural.

## Objetivo

Criar uma experiência mais moderna, institucional, responsiva e segura, mantendo foco em:

- Identidade visual cultural/oficial
- Agenda de concertos
- Página institucional
- Músicos e regência
- Imprensa / EPK
- Apoio cultural
- Contato e booking
- Experiência multilíngue

## Estrutura

- `index.html`: página inicial
- `sobre.html`: história, missão e regência
- `musicos.html`: corpo artístico e mini bios
- `agenda.html`: programação de concertos
- `galeria.html`: memória visual
- `noticias.html`: comunicados e bastidores
- `imprensa.html`: kit de imprensa
- `contato.html`: contato institucional
- `booking.html`: solicitação de apresentações
- `patrocinio.html`: apoio cultural
- `assets/css/`: estilos visuais
- `assets/js/`: scripts de interação
- `assets/data/`: dados estruturados dos músicos
- `assets/img/`: imagens do site

## Manutenção

O conteúdo foi organizado para facilitar atualização de agenda, músicos, imagens, imprensa e páginas institucionais.
