# BSU Site Institucional

Proposta de redesign, melhoria de segurança e migração estática do site da Banda Sinfônica da Unicamp para GitHub Pages.

## Objetivo

Criar uma versão mais moderna, institucional, responsiva e segura do site atual, mantendo foco em:

- Identidade visual cultural/oficial
- Agenda de concertos
- Página institucional
- Músicos e regência
- Imprensa / EPK
- Patrocínio e apoio cultural
- Contato e booking

## Estrutura inicial

- `index.html`: preview da nova landing page
- `docs/relatorio_bsu_melhorias.md`: relatório preliminar de melhorias

## Hospedagem

O site foi preparado para publicação via GitHub Pages usando HTML/CSS estático.
